# Codex Adapter

Use the native Codex skills in `../../skills`:

- `$interchange-message-transformer`
- `$interchange-human-confirmation`
- `$interchange-coding-context`
- `$interchange-flow`
- `$interchange-openspec-workflow`
- `$interchange-connected-mode`
- `$interchange-webhook-sender`

Default to no-config standalone mode. Use `interchange-flow` for project changes that need the OpenSpec-like "agree, implement, archive" loop. Use Connected Mode only when the user explicitly asks for the local Interchange server.
